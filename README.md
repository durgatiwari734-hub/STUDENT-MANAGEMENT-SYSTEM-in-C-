# STUDENT-MANAGEMENT-SYSTEM-in-C-
This project is a Student Management System designed to manage student information efficiently.
It allows adding, updating, deleting, and viewing student records in a structured format. The system is built to simplify the management of student data for schools, colleges, or training institutes.

Key Features
Add, edit, and delete student records
View all student information in an organized way
Search functionality to quickly find a student
User-friendly interface
Simple and functional design

Skills Demonstrated
CRUD operations (Create, Read, Update, Delete)
Data organization and management
